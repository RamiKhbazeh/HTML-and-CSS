# HTML-and-CSS

Welcome to my central repository! This space showcases my journey and progress as a web developer, featuring projects built with HTML, CSS, and modern web frameworks.

## 📁 Included Projects

### 1. The Movie Portal (Bootstrap Project)
A responsive movie theater website built utilizing the **Bootstrap framework**. 
* **Features:** * Fixed navigation bar utilizing HTML bookmarks to jump to specific sections.
  * Modern Bootstrap grid layout featuring "Now Showing" movie cards.
  * Custom images, titles, and descriptions tailored to specific blockbusters.
  * Integration of Bootstrap 'Info Pill Badges' to neatly display movie showtimes.

### 2. Neon Theme Portfolio (HTML & CSS)
A custom-built, colorful, and highly stylized personal portfolio.
* **Features:**
  * **Media Integration:** Includes an auto-playing, muted, looping background video with a stylized text overlay.
  * **Advanced CSS Styling:** Features centered images with CSS filters and rounded edges, plus dynamic image overlay fade effects on hover.
  * **Custom Layouts:** Built using a two-column layout framework with strictly defined heights and widths.
  * **Interactive Elements:** Fixed navigation bar with hover states, dynamic `<p>` tag hover effects, and a fully styled HTML table with borders, padding, and specific row background colors.
  * **Typography:** Centered and bolded footer to cap off the page design.
